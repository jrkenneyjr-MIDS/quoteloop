# QuoteLoop

A free, single-page tool for solo trades and home-service pros: enter an estimate
and get a scheduled, ready-to-send follow-up sequence (SMS + email) plus calendar
reminders — so quotes stop going cold.

Everything runs client-side; no data leaves the browser. This repository hosts the
live MVP via GitHub Pages.

**Live:** https://jrkenneyjr-MIDS.github.io/quoteloop/

Early MVP — built to test whether a lightweight follow-up cadence wins back work for
one-person shops.
